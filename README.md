# Everlasting Chaos

A first-person wave-based survival shooter built in Unity.

This project was developed as part of **Polygon**, a mentored student game development club, during the winter semester of 2020/2021, by a team of 5 people.

## Gameplay

Survive increasingly difficult waves of enemies across multiple levels. Manage your weapons and ammo, eliminate all enemies, and reach the end.

- **Movement** — WASD + Space (jump), Shift (dash)
- **Shoot** — Left Mouse Button
- **Cycle weapons** — Scroll Wheel
- **Drop weapon** — Q

## Features

- 3 levels with wave-based enemy spawning
- Multiple weapons: Pistol, AKM, Rocket Launcher
- AI enemies with NavMesh pathfinding (melee, ranged, and boss variants)
- Health system with UI feedback
- Main menu, pause menu, and options screen

## Tech

- **Engine**: Unity
- **Language**: C#
- **AI**: Unity NavMesh
